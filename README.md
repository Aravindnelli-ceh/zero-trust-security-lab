
# 🔐 Zero Trust Security Lab

A Zero Trust–based security lab that detects phishing, brute-force, and anomalous login behavior using SIEM correlation, DNS filtering, and IDS/IPS—without inspecting encrypted user data.

## 🚀 Features
- SIEM Dashboard
- DNS Filtering Simulation
- IDS/IPS Alerts
- Brute-force Detection
- Behavior-based Risk Scoring

## ⚙️ Tech Stack
- Python (Flask)
- SQLite
- Chart.js

## 🛡️ Security Design
- No plaintext passwords
- Zero Trust architecture
- Behavioral monitoring
